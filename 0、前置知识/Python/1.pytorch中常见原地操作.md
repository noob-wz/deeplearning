x = torch.arange(12).reshape(3,4)
y = torch.arange(12).reshape(3,4)

**绝招 1：切片索引 `z[:]`**
只要你在等号左边写上 `[:]`，就等于告诉 Python：“别动这个箱子，我要修改箱子里的**所有内容**”。

▸ **绝招 2：简写运算符 `+=`**
`y += x` 在 PyTorch 内部是被特殊优化过的，它等价于原地把 `x` 的值加进 `y` 的箱子里。

▸ **绝招 3：下划线后缀 `_`**
PyTorch 里有很多带有特殊下划线的函数，比如 `y.add_(x)`，这个后缀 `_` 在 PyTorch 里的行规就是：**y 原地修改，绝不开辟新内存**。
