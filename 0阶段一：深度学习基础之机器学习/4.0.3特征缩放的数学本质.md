## 1. 宇宙统一公式：平移 + 缩放

特征缩放的本质不是简单的放大缩小，而是**重塑参考系**，因为需要找新的参考原点和尺子单位。无论是哪种方法，底层都共用同一个几何动作：

$$X_{new} = \frac{X - \text{参考原点}}{\text{尺子单位}}$$

* **平移：** 公式里的参考原点对齐到新坐标系的 0 点。从此数据不再是绝对值，而是变成了“偏离中心多少”的相对值。
* **缩放：** 公式的分母意味着定义新坐标轴上的刻度 1

## 2. 两把“手术刀”的终极对决

| 核心维度 | Min-Max 归一化 (Normalization) | Z-score 标准化 (Standardization) |
| :--- | :--- | :--- |
| **数学公式** | $$x' = \frac{x - x_{min}}{x_{max} - x_{min}}$$|$$z = \frac{x - \mu}{\sigma}$$ |
| **参考原点 (0点)** | 最小值 ($x_{min}$) | 均值 ($\mu$) |
| **尺子单位 (长度1)**| 极差 / 区间总长 ($x_{max} - x_{min}$) | 标准差 ($\sigma$) |
| **几何直觉** | **“进度条”：** 你在整个区间里，走到了百分之几？ | **“雷达图”：** 你离大众平均水平，偏离了几个标准差？ |
| **输出边界** | **有严格边界：** 被死死锁在 $[0, 1]$ 之间。 | **无固定边界：** 均值变 0，标准差变 1。可正可负。 |

---

## 3. 极其重要的认知纠错：关于极端值 (Outliers)

很多教材会笼统地说“Z-score 不受极端值影响”，**这是错的！** 准确的表述应该是：**两者都会受极端值影响，只是“死法”不同。**

* **Min-Max 的死法（直接扭曲）：** 极端值会直接撑爆分母（极差）。只要有一个极大值，其余 99% 的正常数据就会被瞬间压缩成一团毫无区分度的肉泥（全部趋近于 0）。
* **Z-score 的死法（中心偏移）：** 极端值会把均值 $\mu$ 拉偏，并撑大标准差 $\sigma$。虽然正常数据也会受牵连，但由于是“均值”在起缓冲作用，正常数据依然能在 0 附近保持一定的正态分布形态，不会被彻底挤死。

---

## 4. 一句话总结 (TL;DR)

> **Min-Max 看“你在整个区间里占几成”，Z-score 看“你离平均水平差几个标准差”。**

## 5. 补充：Z-score输出新的数据集均值为0，标准差为1的证明

### ⚠️ 前置知识提醒

为了推导这个过程，我们需要借用一点“概率统计”里的基本常识。如果你之前没学过也没关系，我用大白话给你补齐这**两个最小必要的数学兵器**：

在数学里，我们把“求平均值”这个动作记作 **$E(\cdot)$**（Expectation，期望），把“求方差”记作 **$Var(\cdot)$**（Variance，方差）。

**兵器 1：求平均值的性质（加减乘除都跟着变）**          
如果把全班同学的分数 $X$ 都乘上一个常数 $a$，再加上一个常数 $b$，那么新的平均分也会跟着乘 $a$ 加 $b$。          

$E(aX + b) = aE(X) + b$


> 💡 **直觉：** 大家的基础分都加 10 分，全班平均分自然也加 10 分。

**兵器 2：求方差的性质（只受乘除影响，不受加减影响）**          

如果把所有数据 $X$ 加减一个常数 $b$，数据的“分散程度（方差）”**不变**。但如果乘上一个常数 $a$，方差会变成原来的 $a^2$ 倍。 

$$Var(aX + b) = a^2 Var(X)$$

> 💡 **直觉：** 全班都加 10 分，大家之间的“分差（分散程度）”是不变的；但如果大家的分数都翻倍，最高分和最低分的差距就会被极度拉开。

### 🚀 开始推导

已知我们的 Z-score 标准化公式是：       
$$Z = \frac{X - \mu}{\sigma}$$
其中：      
* $X$ 是原始数据。
* $\mu$ 是原始数据的平均值（所以 $E(X) = \mu$）。
* $\sigma$ 是原始数据的标准差（所以原始数据的方差 $Var(X) = \sigma^2$）。

为了看清楚，我们把公式拆开，写成 $aX + b$ 的形式：
$$Z = \left(\frac{1}{\sigma}\right)X - \frac{\mu}{\sigma}$$
这里，$a$ 就是 $\frac{1}{\sigma}$，$b$ 就是 $-\frac{\mu}{\sigma}$。

#### 第一步：证明新平均值为 0
我们对新数据 $Z$ 套用“兵器 1”求平均值：
$$E(Z) = E\left(\frac{X - \mu}{\sigma}\right)$$
因为 $\mu$ 和 $\sigma$ 都是算出来的常数（具体的数字），所以我们可以把它们提出来：
$$E(Z) = \frac{E(X) - \mu}{\sigma}$$
刚才说了，原始数据的平均值 $E(X)$ 就是 $\mu$。我们把它代进去：
$$E(Z) = \frac{\mu - \mu}{\sigma} = \frac{0}{\sigma} = 0$$
**🎉 结论 1：** 不管你原来的平均值 $\mu$ 是多少，减去它自己之后，新的平均值铁定变成 0！

#### 第二步：证明新标准差为 1
我们对新数据 $Z$ 套用“兵器 2”求方差：
$$Var(Z) = Var\left(\frac{X - \mu}{\sigma}\right)$$
根据兵器 2，减去常数 $\mu$ 根本不会影响数据的分散程度，可以直接扔掉。而除以常数 $\sigma$（也就是乘以 $\frac{1}{\sigma}$），提出来时要平方：
$$Var(Z) = \left(\frac{1}{\sigma}\right)^2 Var(X) = \frac{Var(X)}{\sigma^2}$$
刚才也说了，原始数据的方差 $Var(X)$ 就是 $\sigma^2$。我们把它代进去：
$$Var(Z) = \frac{\sigma^2}{\sigma^2} = 1$$
既然新方差是 1，那么新标准差（方差的平方根 $\sqrt{1}$）自然也是 1！     
**🎉 结论 2：** 不管原来的尺子刻度多大，除以它自己之后，新的尺子刻度铁定变成 1！
