第一章

线性回归：用一根木棍拟合散点

想象你收集了一批数据：房子面积和对应价格。把这些点画在坐标系里，你会发现它们大致沿着一条斜线分布。线性回归的任务，就是找到那根"最合适"的线。

设计者的第一个问题：什么叫"最合适"？必须先定义一个衡量标准，才能谈"最优"。这个标准就是损失函数。

模型长什么样

线性回归的预测非常直白，就是一条直线方程：

ŷ = w · x + b
// ŷ = 预测值，x = 输入特征，w = 权重（斜率），b = 偏置（截距）

这里只有两个参数需要学习：w（斜率，决定直线倾斜角度） 和 b（截距，决定直线上下位置）。整个训练过程就是找到最好的 w 和 b。

比喻解释：把每个样本点当作钉在木板上的图钉，直线是一根木棍，每个图钉和木棍之间用弹簧相连。弹簧越长（误差越大），拉力越大。木棍会在所有弹簧的合力下自动旋转、平移，直到"所有弹力之和最小"——这就是模型训练的本质。

MSE vs MAE 的区别：

· MSE（均方误差）下：弹力的大小（模型对单个样本的敏感度） = Loss对预测值的导数 = 2(预测值-真实值) ，离得远的点惩罚急剧增大，对异常值非常敏感。

· MAE（平均绝对误差）下：弹力的大小（模型对单个样本的敏感度） = Loss对预测值的导数 = 正负1，所有样本点对模型的拉力一致，对异常点的抗干扰能力强

工程上默认用 MSE，因为它处处可导，梯度下降更顺畅。
// MSE 损失函数
L = (1/n) · Σ (ŷᵢ − yᵢ)²

// 展开：预测值 ŷ = wx + b
L = (1/n) · Σ (wxᵢ+b - yᵢ)²

🎯 梯度下降：怎么调整木棍

有了损失函数，下一个问题是：如何系统地调整 w 和 b 让损失变小？

答案是对损失函数求导。导数告诉我们：当前 w 和 b 在"损失地形"的哪个方向上是下坡方向。沿着下坡方向走，损失就会减小——这就是梯度下降。

// 对 w 求偏导（"敏感度 × 力矩" 的直觉）
∂L/∂w = (2/n) · Σ (ŷᵢ - yᵢ) · xᵢ

// 对 b 求偏导（只是敏感度之和）
∂L/∂b = (2/n) · Σ (ŷᵢ - yᵢ)

// 更新规则（α = 学习率）
w ← w - α · ∂L/∂w
b ← b - α · ∂L/∂b
根据链式法则（针对所有模型）

关于 w 的导数 = "L 关于 ŷ 的导数" 乘以 "ŷ 关于 w 的导数"。所以在线性回归模型下：
  • L 关于 ŷ 的导数 → 就是弹力的大小（模型对单个样本点的敏感度）
  • ŷ 关于 w 的导数 → 就是特征 x（力矩）

所以，单个样本对模型最终的影响（真实破坏力） = 对权重的导数 = 弹簧弹力大小(L 关于 ŷ 的导数) * 力矩(x)。

总结就是：一个样本点对模型的最终影响，不仅取决于它错得有多离谱（弹簧拉力），还取决于它站在什么位置（杠杆有多长）！>

第二章

为什么线性回归不能做分类？

假设我们要判断一封邮件是否是垃圾邮件（是=1，否=0）。如果直接用线性回归 ŷ = wx + b 来预测标签 y，会遇到三个致命问题。下面的交互演示将同时体现这三个问题：

致命问题一：输出没有边界
概率必须在 [0, 1] 之间，但线性回归的输出 ŷ = wx + b 可以是任意实数——它可能输出 −0.5 或 1.3，这在"概率"语境下毫无意义。图中红色区域标出了 ŷ 超出 [0,1] 的无意义地带。
致命问题二：对异常值极度敏感
MSE 损失下，一个极远的点产生的梯度 = 敏感度 2(ŷ−y) × 力矩 x，两者都随距离增大。这个巨大的梯度会强行把整条直线拉偏，导致原本分好类的样本全部阵亡。拖动滑块加入极端点，观察分界线如何被"绑架"。
致命问题三：损失函数不匹配
分类问题是"对了就是对了，错了就是错了"，不适合用数值大小来惩罚。MSE 无法区分"差一点分对"和"完全分错"在分类意义上的天壤之别 ———— 它只看数值差距，不看分类结果。

❷ 致命问题二：分界线被极端点"绑架"，疯狂右移，导致左边原本正确的分类全部阵亡（误判数飙升）
❸ 致命问题三：MSE 只看数值差距，无法告诉模型"这些误判在分类意义上是灾难性的"
设计者的思路：我们需要一个函数，能把任意实数 z 压缩到 (0,1) 范围内，而且在 z=0 处有一个自然的分界点。有没有这样的函数？——有，它叫 Sigmoid。
第三章

Sigmoid：把任意数值压缩成概率

这一章要回答最容易困惑的问题：决策边界是 sigmoid 图上的一个点（z=0），但在分类图里却变成了一条线（wx+b=0）。它们怎么关联？

σ(z) = 1 / (1 + e-z)

// z 可以是任意实数：-∞ 到 +∞
// σ(z) 的输出：永远在 (0, 1) 之间
// σ(0) = 0.5 ← 这是关键：z=0 是自然分界点

🔗 联动演示：从特征 x 到概率 p 的完整因果链

⚠️ 阅读本图前，务必先理解这条因果链：

① σ(z) 这条 S 型曲线本身永远不变——它只是一张固定的"z→概率"查分表（下方图横轴是 z）
② 改变 w 和 b，改变的是映射规则 z = wx+b："同一个特征值 x，被映射到不同的 z 值"
③ 决策边界永远在 z = 0（因为 σ(0)=0.5），但满足 z=0 的那个特征值 x* = −b/w 会随 w、b 变化
④ 所以拖动 w、b 时，你看到的是：上方特征空间的橙色分界线在移动，而下方 S 曲线纹丝不动

移动的不是 sigmoid 曲线，而是"哪个特征值 x 会被映射到 z=0"。

🎯 核心顿悟：拖动 w 或 b 时，下方的 S 曲线始终不变。变化的是上方的橙色分界线 x*=−b/w：w 和 b 改变了映射规则 z=wx+b，使得满足 z=0 的那个特征值 x* 发生了位移。sigmoid 不动，动的是"谁会被映射到 z=0"。
因果顺序（一定要记住这个方向）：
① 我们用 sigmoid 做概率输出 → ② 用 p=0.5 作为分类阈值 → ③ p=0.5 对应 σ(z)=0.5 → ④ z=0 → ⑤ wx+b=0

所以：
· z=0 是 sigmoid 图上的一个永恒不变的及格分——无论 w、b 怎么变，σ(0)=0.5 永远成立
· wx+b=0 是这个及格分投射回现实特征空间里，划出的一道楚河汉界——满足"z=0"的所有 x 构成的集合
· 一维时（只有一个特征 x），这道界限是一个点 x=−b/w
· 二维时（两个特征 x₁, x₂），w₁x₁+w₂x₂+b=0 是平面上的一条线

wx+b=0 不是我们主观规定的，它是选用 sigmoid 并以 p=0.5 为阈值的自然结果。逻辑回归的训练，本质就是寻找最好的 w 和 b，使得 z=wx+b=0 这条分界线恰好把两类样本分开。
改变 w 和 b 分别在做什么（精确表述）：
· 改变 w → 改变映射 z=wx+b 中 x 的"放大倍率"。注意区分两个坐标系：
    · 以 z 为横轴画 σ(z)：曲线形状永远不变，因为 σ 函数本身不含 w
    · 以 特征 x 为横轴画 σ(wx+b)：|w| 越大，过渡带越窄、曲线越陡——因为 x 的微小变化被 w 放大后，z 变化剧烈，分类越"非黑即白"
· 改变 b → 整体平移 z 值，等价于把决策边界 x*=−b/w 向左或向右移动
· w 和 b 是模型训练学到的，一旦确定，映射 z=wx+b 就定死了，决策边界也随之锁死
第四章

逻辑回归：完整拼图

我们有了把实数变成概率的 Sigmoid，也有了分类的楚河汉界。现在，让我们把前三章的零件全部组装起来，这就是逻辑回归的真面目。

🗺️ 完整流程图

逻辑回归 · 前向传播 + 反向传播
z = w₁x₁ + w₂x₂ + ··· + wₙxₙ + b
// 向量形式：z = wᵀx + b
// z 是"证据强度"：越大越倾向类别1，越小越倾向类别0，z=0是中立

z 是任意实数，它是所有特征的"加权投票"结果。每个 wᵢ 代表特征 xᵢ 在判断中的重要程度和方向。

p = σ(z) = 1 / (1 + e⁻ᶻ)
// p = 样本属于类别1的概率
// 决策：p ≥ 0.5 → 类别1 · p < 0.5 → 类别0

Sigmoid 把"证据强度"z 转换成"概率"p。这不只是为了把数字强行塞进 0~1 的凑公式技巧——在统计学里，当我们想用抛硬币（非黑即白）的逻辑来拟合现实规律时，数学推导出的终极自然公式，刚好就是 Sigmoid。

// 交叉熵损失（单个样本）
L = −[y·log(p) + (1−y)·log(1−p)]

// y=1 时：L = −log(p) p→0 时 L→∞
// y=0 时：L = −log(1−p) p→1 时 L→∞
交叉熵损失的行为
⚠️ 你之前只写了 −log(p)，这只对 y=1 成立！
完整形式用 y 自动"切换"：y=1时只留第一项，y=0时只留第二项。
两种错误都会被无穷大惩罚，这正是交叉熵的核心设计。
为什么不用 MSE？逻辑回归用 MSE 会导致损失曲面非凸（有很多局部最小），梯度下降容易卡住。交叉熵在 sigmoid 输出下保证损失曲面是凸的，有唯一全局最小值。

真实的梯度下降不是直接操作 p，而是操作 w 和 b。我们需要 ∂L/∂w，这要通过三层链式法则展开：

损失→概率的敏感度
∂L/∂p
×
Sigmoid 传导率
∂p/∂z
×
z→w 的力矩
∂z/∂w = x
// 第一层：损失对概率的敏感度
∂L/∂p = −y/p + (1−y)/(1−p)

// 第二层：Sigmoid 对 z 的导数（自带的漂亮性质）
∂p/∂z = p · (1−p)

// 第三层：z 对 w 的导数（力矩）
∂z/∂w = x

// 三层相乘，神奇地化简！（过程需要代数展开）
∂L/∂w = (p − y) · x
化简结果惊人地优雅：三层链式法则的乘积 [−y/p+(1−y)/(1−p)] × [p(1−p)] × x，化简后就是 (p−y)·x。这不是巧合，是 sigmoid + 交叉熵"天生一对"的数学体现——两者搭配会消去复杂项。
// 全数据集平均梯度（和线性回归形式惊人一致！）
∂J/∂w = (1/n) · Σ (pᵢ − yᵢ) · xᵢ
∂J/∂b = (1/n) · Σ (pᵢ − yᵢ)

w ← w − α · ∂J/∂w
b ← b − α · ∂J/∂b
线性回归 vs 逻辑回归梯度对比：
线性回归：∂L/∂w = (ŷ−y)·x  ← 敏感度 2(ŷ−y) × 力矩 x（省略常数2）
逻辑回归：∂L/∂w = (p−y)·x  ← 三层链式法则化简后的结果
形式完全一样，只是"误差"的来源不同。这是广义线性模型 (GLM) 的统一结构。

⚖️ 比喻：逻辑回归的"态度考官"vs 线性回归的"无脑弹簧"

用链式法则精确对比两种模型的"力度"：

线性回归（无脑弹簧）：
· 敏感度 ∂L/∂ŷ = 2(ŷ−y)：损失对预测值的导数——离真值越远，惩罚越大，与几何距离成正比
· 最终影响 ∂L/∂w = ∂L/∂ŷ × ∂ŷ/∂w = 2(ŷ−y) · x：敏感度 × 力矩

逻辑回归（态度考官）：
· 敏感度 ∂L/∂p = −y/p + (1−y)/(1−p)：损失对概率的导数——错得越自信，惩罚越剧烈
· 经 sigmoid 传导后，最终影响 ∂L/∂w = ∂L/∂p × ∂p/∂z × ∂z/∂w = (p−y) · x：化简后的优雅结果

根本差异：线性回归的敏感度 ∝ 几何距离；逻辑回归的敏感度 ∝ 概率错误程度。

精确对比——两种"敏感度"的根本差异：
· 线性回归：敏感度 ∂L/∂ŷ = 2(ŷ−y)，与预测值和真值的几何差距成正比
· 逻辑回归：敏感度 ∂L/∂p = −y/p+(1−y)/(1−p)，取决于概率的错误自信程度
· 两者的最终影响（∂L/∂w）都要再乘以力矩（∂ŷ/∂w 或 ∂z/∂w），但敏感度的来源本质不同

· 一个点在决策边界旁（几何距离近），若被错误分类且模型极其自信（p≈1，y=0），(p−y)≈1，惩罚极大
· 一个点离决策边界很远，但已被正确分类且很自信（p≈1，y=1），(p−y)≈0，梯度接近 0——不再更新
💡 进阶认知：不仅要在乎"对不对"，还要在乎"有多确定"
如果一个样本被正确分类，但模型只给出了 0.51 的概率，此时 (p−y) 依然有近 0.49 的误差，梯度依然会产生拉力！这会推动模型继续调整，直到它敢于给出 0.99 的极高置信度为止。这就是交叉熵的魅力——它追求的是"极其自信地做对"。

第五章

对比总结

线性回归

项目内容
任务预测连续数值
输出任意实数 ŷ
损失函数MSE: (ŷ−y)²
敏感度 ∂L/∂ŷ2(ŷ−y) ← 与几何距离成正比
最终影响 ∂L/∂w(ŷ−y)·x ← 敏感度 × 力矩
激活函数无（恒等函数）
比喻无脑弹簧：力∝几何距离

逻辑回归

项目内容
任务预测类别（分类）
输出概率 p ∈ (0,1)
损失函数交叉熵: −[y·log(p)+…]
敏感度 ∂L/∂p−y/p+(1−y)/(1−p) ← 与概率错误自信程度相关
最终影响 ∂L/∂w(p−y)·x ← 三层链式法则化简
激活函数Sigmoid σ(z)
比喻态度考官：力∝概率错误程度
统一视角：广义线性模型 (GLM)
核心都是 z = wx+b，区别只是套了什么"激活函数"：
· 线性回归 → 恒等激活 → 预测连续值
· 逻辑回归 → Sigmoid 激活 → 预测概率
· 神经网络 → ReLU/tanh/… 激活 → 学习更复杂的映射
神经网络就是把这个思路堆叠多层。

梯度公式的对称美

// 线性回归
∂L/∂w = (1/n) · Σ (ŷᵢ - yᵢ) · xᵢ

// 逻辑回归（形式完全一样！只是误差定义不同）
∂J/∂w = (1/n) · Σ (pᵢ - yᵢ) · xᵢ

// 统一形式：梯度 = 误差 × 特征，这是最重要的直觉
🎯 记住这一句话：训练 = 找让损失 J(w,b) 最小的 w 和 b。梯度下降 = 算下坡方向，迈一小步。不管模型多复杂，本质都是这一件事。
