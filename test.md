## Week 15 · 应力、应变与本构关系（直觉层）

### Day 1 ｜ 应力的物理含义

**理论目标**
- [ ] 🧠 L2 理解 ⭐：应力 = 单位面积上的内力

```
一根杆件截面积 A，受拉力 F
应力 σ = F / A（单位：MPa = N/mm²）

直觉：同样的力拉细杆比粗杆"更吃力"——因为细杆应力更大
```

- [ ] 🧠 L2 理解：应力不是一个数，而是一个**张量**（2D 有 3 个分量，3D 有 6 个）

```
2D 应力状态：
σxx = x 方向的正应力（拉/压）
σyy = y 方向的正应力
σxy = 剪应力（让材料"滑动"的力）
```

- [ ] 🧠 L2 理解：为什么要理解这个——你的 GNN 输出的 σxx, σyy, σxy 就是这三个分量，不是随便的三个数

### Day 2 ｜ 应变的物理含义

**理论目标**
- [ ] 🧠 L2 理解：应变 = 变形量 / 原始长度（无量纲）

```
一根杆原长 L，拉伸后变长 ΔL
应变 ε = ΔL / L

直觉：ε = 0.01 意味着伸长了 1%
```

- [ ] 🧠 L2 理解：2D 应变分量

```
εxx = ∂ux/∂x   x 方向的拉伸/压缩
εyy = ∂uy/∂y   y 方向的拉伸/压缩
εxy = ½(∂ux/∂y + ∂uy/∂x)   剪切变形
```

- [ ] 🧠 L2 理解：你在 PINN 里用 autograd 算的 `∂u/∂x` 就是应变——现在你知道它的物理含义了
- [ ] 🧠 L1 知道：工程应变 vs 对数应变——大变形（冲压）时用对数应变 ε = ln(1 + ΔL/L)

### Day 3 ｜ 本构方程——应力和应变的关系

**理论目标**
- [ ] 🧠 L2 理解 ⭐：本构方程 = 描述材料"脾气"的方程——同样的变形，不同材料会产生不同的力

```
线弹性（胡克定律）：σ = E × ε
  E = 弹性模量（材料的"刚度"）
  钢 E ≈ 200 GPa（很硬）
  铝 E ≈ 70 GPa（较软）
  橡胶 E ≈ 0.01 GPa（非常软）
```

- [ ] 🧠 L2 理解：2D 平面应力本构（你在 Week 29 会嵌入的公式）

```
σxx = E/(1-ν²) × (εxx + ν·εyy)
σyy = E/(1-ν²) × (εyy + ν·εxx)
σxy = E/(2(1+ν)) × εxy

其中 ν = 泊松比（拉长时变细的程度，钢约 0.3）
```

- [ ] 🧠 L2 理解：泊松效应的直觉——拉一根橡皮筋，它会变长但也会变细，ν 就是描述"变细多少"的参数

**实践目标**
- [ ] 💻 L2 能照写：用 Python 实现 2D 本构方程

```python
def hooke_2d(eps_xx, eps_yy, eps_xy, E, nu):
    coeff = E / (1 - nu**2)
    sigma_xx = coeff * (eps_xx + nu * eps_yy)
    sigma_yy = coeff * (eps_yy + nu * eps_xx)
    sigma_xy = E / (2 * (1 + nu)) * eps_xy
    return sigma_xx, sigma_yy, sigma_xy
```

- [ ] 💻 L2 能照写：代入具体数字验证——如果 εxx=0.001, εyy=0, E=200GPa, ν=0.3，σxx=?

### Day 4 ｜ 主应力与 von Mises 应力

**理论目标**
- [ ] 🧠 L2 理解 ⭐：主应力 = 把应力张量"旋转"到没有剪应力的方向

```
2D 主应力公式：
σ₁,₂ = (σxx + σyy)/2 ± √[((σxx - σyy)/2)² + σxy²]

σ₁ = 最大主应力（最大的拉力方向）
σ₂ = 最小主应力
```

- [ ] 🧠 L2 理解 ⭐：von Mises 等效应力——把复杂的多轴应力状态简化成一个数

```
2D von Mises：σ_vm = √(σxx² - σxx·σyy + σyy² + 3·σxy²)

直觉：σ_vm 是衡量"材料有多危险"的综合指标
工程师用它来判断：σ_vm > 屈服强度 → 材料开始塑性变形 → 危险！
```

- [ ] 🧠 L2 理解：这就是你 FEM 后处理里看到的"应力云图"——颜色深 = von Mises 高 = 应力集中

**实践目标**
- [ ] 💻 L2 能照写：用 Python 算主应力和 von Mises

```python
def von_mises_2d(sxx, syy, sxy):
    return np.sqrt(sxx**2 - sxx*syy + syy**2 + 3*sxy**2)

def principal_stress_2d(sxx, syy, sxy):
    avg = (sxx + syy) / 2
    diff = np.sqrt(((sxx - syy) / 2)**2 + sxy**2)
    return avg + diff, avg - diff  # σ₁, σ₂
```

### Day 5–6（周末）｜ 应力-应变曲线与屈服

**理论目标**
- [ ] 🧠 L2 理解 ⭐⭐：典型金属的应力-应变曲线

```
应力 σ
  │          ╱ 断裂
  │        ╱
  │      ╱
  │    ╱ 硬化段（塑性变形，越拉越硬但速度变慢）
  │──╱ ← 屈服点 σ_y（材料开始"永久变形"的临界点）
  │╱ 弹性段（卸载后完全恢复，斜率 = E）
  └──────────────── 应变 ε
```

- [ ] 🧠 L2 理解：弹性段——像弹簧，拉多少回多少，σ = E·ε
- [ ] 🧠 L2 理解：屈服点 σ_y——超过这个应力后材料开始"永久变形"
- [ ] 🧠 L2 理解 ⭐：屈服准则（von Mises 屈服）

```
如果 σ_vm ≥ σ_y → 材料进入塑性变形
这就是为什么 von Mises 应力这么重要——它直接决定材料是否"坏了"
```

- [ ] 🧠 L2 理解：为什么面试一定会问这个——汽车冲压/碰撞全是超过屈服点的大变形问题

**实践目标**
- [ ] 💻 L2 能照写：画一条典型的应力-应变曲线（用 matplotlib）

```python
eps = np.linspace(0, 0.3, 200)
sigma_y = 350  # MPa 屈服强度
E = 200e3      # MPa 弹性模量
K = 600        # MPa 硬化系数
n = 0.2        # 硬化指数

# 弹性段
sigma_elastic = E * eps
# 塑性段（幂指数硬化）
eps_plastic = eps - sigma_y / E
eps_plastic = np.maximum(eps_plastic, 0)
sigma_plastic = sigma_y + K * eps_plastic ** n

sigma = np.where(eps < sigma_y / E, sigma_elastic, sigma_plastic)
plt.plot(eps, sigma)
plt.xlabel('Strain ε'); plt.ylabel('Stress σ (MPa)')
plt.axhline(sigma_y, ls='--', color='r', label=f'Yield σ_y={sigma_y} MPa')
plt.legend()
```

## Week 16 · 弹塑性与工程常识

**理论目标**
- [ ] 🧠 L2 理解 ⭐：一张表搞清两种行为

| | 弹性变形 | 塑性变形 |
|---|---------|---------|
| 应力范围 | σ < σ_y | σ ≥ σ_y |
| 可恢复？ | 卸载后完全恢复 | 卸载后有永久变形 |

- [ ] 🧠 L2 理解：汽车行业大部分仿真问题都涉及塑性——冲压成形、碰撞安全、疲劳

### Day 2 ｜ 硬化模型

**理论目标**
- [ ] 🧠 L2 理解：超过屈服点后，材料不是"一下子断掉"，而是"越拉越硬但硬得越来越慢"


```python
from odbAccess import openOdb

odb = openOdb('plate_hole.odb')
step = odb.steps['Step-1']
frame = step.frames[-1]  # 最后一帧

# 节点坐标
instance = odb.rootAssembly.instances['PLATE-1']
nodes = {}
for node in instance.nodes:
    nodes[node.label] = [node.coordinates[0], node.coordinates[1]]

# 节点位移
disp_field = frame.fieldOutputs['U']
displacements = {}
for val in disp_field.values:
    displacements[val.nodeLabel] = [val.data[0], val.data[1]]

# 单元应力（取积分点平均值或节点外推值）
stress_field = frame.fieldOutputs['S']
stresses = {}
for val in stress_field.values:
    # val.data = [S11, S22, S33, S12, ...] 取决于维度
    stresses[val.elementLabel] = list(val.data[:3])  # Sxx, Syy, Sxy

odb.close()
```

### 新增八股题（Q23–Q30）

**Q23：应力和力有什么区别？**
```
力 = 作用在物体上的外部作用（单位 N）
应力 = 单位面积上的内力（单位 MPa = N/mm²）
同样 100N 的力，作用在 1mm² 上是 100MPa，作用在 100mm² 上只有 1MPa
工程上关心的是应力而不是力——因为应力决定材料是否破坏
```

**Q24：von Mises 应力是什么？为什么不直接看 σxx？**
```
实际受力通常是多轴的（同时有拉伸+剪切）
von Mises 把复杂的多轴状态简化成一个等效标量
物理含义：等效于单轴拉伸时产生同样"危险程度"的应力
用它判断屈服：σ_vm ≥ σ_y → 进入塑性
```

**Q25：弹性和塑性的区别？汽车仿真中哪种更常见？**
```
弹性：卸载恢复，σ=Eε，线性
塑性：卸载后永久变形，σ=Kε^n，非线性
汽车中：冲压/碰撞 = 塑性为主；NVH = 弹性为主
你做的项目如果只涉及弹性，面试时要能说清楚"扩展到塑性需要换本构方程"
```

**Q26：什么是本构方程？你嵌入的是哪个？**
```
本构方程 = 材料的"性格说明书"——告诉你"这种材料受力后会怎么变形"
我嵌入的是线弹性本构（胡克定律的 2D 平面应力形式）
σxx = E/(1-ν²) × (εxx + ν·εyy) ...
扩展到塑性：换成幂指数模型 σ = K·ε^n
```

**Q27：你的训练数据来自哪个软件？数据格式是什么？**
```
来自 Abaqus（/FEniCS）
输入格式：.inp 文件（节点坐标 + 单元连接 + 材料 + 边界条件）
输出格式：.odb 文件（位移/应力/应变场）
我用 Python 脚本提取 .odb 中的数据，转成 NumPy 数组，再转成 PyG 图数据
```

**Q28：Abaqus 的建模流程是什么？**
```
Part → Property → Assembly → Step → Load → BC → Mesh → Job → Visualization
（零件 → 材料 → 装配 → 分析步 → 载荷 → 边界条件 → 网格 → 提交 → 后处理）
每一步都可以用 Python 脚本自动化
```

**Q29：你知道什么是应力集中吗？为什么你的模型在孔边误差大？**
```
应力集中 = 几何突变处（如孔、缺口、圆角）应力急剧升高
孔边的 von Mises 可能是远处的 3 倍（应力集中系数 ≈ 3）
模型误差大的原因：
1. 孔边梯度急变，有限层消息传递的感受野不够
2. 训练数据中这个区域的"难度"最高
改进思路：孔边网格加密 / 增加消息传递层数 / 加局部注意力机制
```

**Q30：如果数据来自不同的 FEM 软件（如 LS-DYNA 和 Abaqus），能混用吗？**
```
可以，但需要注意：
1. 单元类型可能不同（需要统一或分别处理）
2. 应力/应变的定义可能有差异（积分点位置、输出格式）
3. 坐标系和单位可能不同（必须统一）
核心：数据转成统一的 graph 格式后，GNN 不关心数据来自哪个软件
这也是 GNN 方案的一个优势——数据格式无关性
```
