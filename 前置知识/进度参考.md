## 🥇 最推荐：Fast.ai《Practical Deep Learning for Coders》

**官网：** fast.ai
**逻辑：自上而下（先跑通代码，再理解原理）**

```
Part 1 - 实战入门（约7周）
├── Lesson 1: 图像分类入门（直接上手训练模型）
├── Lesson 2: 模型部署 & 避免常见错误
├── Lesson 3: 神经网络 & 基础数学
├── Lesson 4: 自然语言处理入门
├── Lesson 5: 从零构建神经网络（Tabular）
├── Lesson 6: 随机森林 & 特征理解
└── Lesson 7: 协同过滤 & 嵌入层

Part 2 - 深入原理（约7周）
├── 手写 Diffusion Model
├── 手写 ResNet / U-Net
├── 手写训练框架（理解反向传播底层）
└── 稳定扩散模型原理
```

✅ 优点：新手最友好，第一节课就能跑通图像分类；工程感极强
⚠️ 注意：CV特化内容需要自己延伸

## 🥈 体系最完整：DeepLearning.AI《Deep Learning Specialization》

**平台：** Coursera（吴恩达主讲）
**逻辑：自下而上（先打基础再实战）**

```
Course 1: 神经网络与深度学习基础
├── 逻辑回归 → 浅层神经网络 → 深层网络
└── 向量化、激活函数、反向传播推导

Course 2: 改善深层神经网络
├── 正则化、Dropout、Batch Norm
└── 优化算法（Adam、RMSprop 等）

Course 3: 结构化机器学习项目
├── 误差分析、数据集分配策略
└── 端到端学习思路

Course 4: 卷积神经网络（CV核心！）
├── CNN基础 → 经典网络（ResNet/VGG/Inception）
├── 目标检测（YOLO、R-CNN 系列）
├── 人脸识别
└── 神经风格迁移

Course 5: 序列模型
└── RNN / LSTM / Transformer 入门
```

✅ 优点：Course 4 对你的方向直接有用；体系完整不跳跃
⚠️ 注意：偏公式推导，编程作业用 NumPy，工程感稍弱

## 🗺️ 给你的学习路线建议

```
阶段一（1-2个月）基础搭建
  → d2l.ai 第1-2部分 + Fast.ai Part1 前3课
  → 目标：能独立跑通一个图像分类任务

阶段二（2-3个月）CV核心
  → DeepLearning.AI Course 4（卷积网络）
  → d2l.ai 第三部分
  → 目标：理解 YOLO、ResNet，能做目标检测

阶段三（3-4个月）工程实战
  → Udacity 自动驾驶课程 or 自选项目
  → 做：车道线检测 / 行人检测 / 语义分割
  → 目标：GitHub 上有2-3个可以展示的项目
```

替代方案
```
阶段零（第 1-2 个月）打地基  ← 原规划缺失的部分
  → Python 基础：推荐《Python编程：从入门到实践》前半部分
  → 数学直觉：3Blue1Brown 的《线性代数的本质》《微积分的本质》（B站有）
  → 目标：能独立写 Python 脚本，对矩阵乘法有直觉感

阶段一（第 3-4 个月）建立信心
  → 只做 Fast.ai Part 1（全部7课）
  → d2l.ai 作为"字典"遇到不懂的概念再去查，不要从头线性阅读
  → 目标：跑通图像分类，部署一个简单模型

阶段二（第 5-7 个月）CV 理论打扎实
  → DeepLearning.AI Course 4（卷积网络，这门最关键）
  → d2l.ai 目标检测部分配合看
  → 目标：真正理解 YOLO / ResNet，手写过一个目标检测 pipeline

阶段三（第 8-12 个月）工程实战
  → 用开源数据集做项目：车道线检测 / 行人检测 / 语义分割
  → 替代 Udacity：GitHub 上的 Apollo / OpenCV 示例 + Kaggle 比赛
  → 目标：GitHub 上 2-3 个可展示项目 + 能读懂论文
```
