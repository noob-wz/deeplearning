第一天：Preparation 影片5（BN预习）

第二天：Week 2 正课视频（前半）
  对应内容：
  ├─ 局部最小值与鞍点（梯度流动直觉）
  └─ Batch与Momentum优化器

第三天：Week 2 正课视频（后半）
  对应内容：
  ├─ Adam自适应学习率 ← 吴恩达没讲，CV训练标配
  ├─ 损失函数选择（为什么分类不用MSE）

第四天：Basic theory补充视频

第五天：Extra - Beyond Adam part 1 + part 2

第六天：自己动手

## 第四阶段｜吴恩达第二门课 第3周
**预计时间：8-10天（刻意放慢）**

> 目标：建立系统性调试能力，这是李宏毅覆盖不到的部分

```
第一部分（2-3天）：偏差与方差诊断
  → 诊断偏差与方差
  → 正则化与偏差/方差
  → 建立基准性能水平
  → 学习曲线
  重点：不只是看视频，要自己画学习曲线

第二部分（2天）：神经网络中的偏差与方差
  → 大模型+正则化的现代解决思路
  → 可选实验：偏差与方差诊断（必须做）

第三部分（2天）：机器学习开发流程
  → 迭代循环
  → 误差分析（手动检查错误样本）
  → 迁移学习 ← CV几乎都用迁移学习，这节很重要

第四部分（2-3天）：实战练习
  ⚠️ 这个阶段最容易"听懂但不会用"
  做这件事来验证自己真的懂了：
  → 找一个简单的CV任务（CIFAR-10图像分类）
  → 跑完整训练
  → 自己画学习曲线
  → 判断是过拟合还是欠拟合
  → 根据判断结果调整模型
```

**为什么不用李宏毅替换：** 李宏毅只能覆盖偏差/方差的概念，但吴恩达这周的核心价值是**系统性调试方法论**——误差分析、迁移学习实战逻辑、学习曲线判读，这些李宏毅是零散的，吴恩达是完整的。
