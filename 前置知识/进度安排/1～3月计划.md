### Day 4 ｜ 标准训练循环模板 ⭐（全路线图最重要的代码）

**理论目标**
- [ ] 🧠 L2 理解：训练循环 5 步每一步在干什么：

| 步骤 | 代码 | 含义 |
|------|------|------|
| ① 前向传播 | `output = model(x)` | 数据过模型得到预测 |
| ② 算损失 | `loss = criterion(output, y)` | 预测和真实值算距离 |
| ③ 清旧梯度 | `optimizer.zero_grad()` | 清空上一轮残留的梯度 |
| ④ 反向传播 | `loss.backward()` | 自动算出新梯度 |
| ⑤ 更新参数 | `optimizer.step()` | 沿梯度方向更新参数 |

- [ ] 🧠 L2 理解：为什么需要 `zero_grad()`？因为 PyTorch 默认会累加梯度，不清空就乱了

**实践目标**
- [ ] 💻 L3 能默写 ⭐：不看任何参考，能从空白文件写出以下模板

```python
model = SimpleNet()
criterion = nn.BCELoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.01)

for epoch in range(num_epochs):
    for batch_x, batch_y in dataloader:
        output = model(batch_x)
        loss = criterion(output, batch_y)
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
```

- [ ] 💻 L2 能照写：加上每个 epoch 打印 loss 的逻辑

**⚠️ 重要**：这段代码会伴随你一整年。本周必须做到"默写"级别。建议每天早上起来默写一遍，连写 3 天就刻入肌肉记忆。

```python
    def __init__(self):
        super().__init__()
        self.conv1 = nn.Conv2d(1, 16, 3, padding=1)
        self.conv2 = nn.Conv2d(16, 32, 3, padding=1)
        self.pool = nn.MaxPool2d(2)
        self.fc1 = nn.Linear(32 * 7 * 7, 128)
        self.fc2 = nn.Linear(128, 10)

    def forward(self, x):
        x = self.pool(torch.relu(self.conv1(x)))
        x = self.pool(torch.relu(self.conv2(x)))
        x = x.view(x.size(0), -1)
        x = torch.relu(self.fc1(x))
        return self.fc2(x)
```

### Day 4 ｜ Early Stopping

**理论目标**
- [ ] 🧠 L2 理解：Val Loss 连续 N 个 epoch 不下降 → 再训下去只会更过拟合 → 该停了
- [ ] 🧠 L1 知道：patience = 容忍多少个 epoch 没有改进

```python
class EarlyStopping:
    def __init__(self, patience=5):
        self.patience = patience
        self.counter = 0
        self.best_loss = float('inf')

    def should_stop(self, val_loss):
        if val_loss < self.best_loss:
            self.best_loss = val_loss
            self.counter = 0
        else:
            self.counter += 1
        return self.counter >= self.patience
```

## Week 10 · 误差分析（Error Analysis）

### Day 1 ｜ 误差分析方法论

**理论目标**
- [ ] 🧠 L2 理解 ⭐：误差分析四步法
  1. 收集所有预测错误的样本
  2. 逐一检查，标注错误原因
  3. 统计各原因的占比
  4. 优先解决占比最高的那个
- [ ] 🧠 L2 理解：Ceiling Analysis — 即使完美解决这类错误，最多能提升多少？
- [ ] 🧠 L2 理解：这套方法的通用性 — 检测、分割、NLP 都适用

### Day 2 ｜ 提取错误样本

**实践目标**

```python
wrong_samples = []
model.eval()
with torch.no_grad():
    for images, labels in test_loader:
        outputs = model(images)
        _, predicted = torch.max(outputs, 1)
        mask = predicted != labels
        for img, pred, true in zip(images[mask], predicted[mask], labels[mask]):
            wrong_samples.append((img, pred.item(), true.item()))
```
