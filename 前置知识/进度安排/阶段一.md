### Day 2 ｜ 参数更新公式

**理论目标**
- [ ] 🧠 L3 推导：参数更新

```
θ_(k+1) = θ_k − η · ∂L/∂θ
```

  - θ：参数（w 或 b）
  - η：学习率，控制每步走多远
  - ∂L/∂θ：损失对参数的偏导数

- [ ] 🧠 L2 理解：学习率太大 → 震荡/发散；太小 → 收敛极慢

**实践目标**
- [ ] 💻 L3 能默写：线性回归的梯度推导

```python
# y_pred = w * x + b
# L = (1/N) * Σ(y_pred - y_true)²
# ∂L/∂w = (2/N) * Σ(y_pred - y_true) * x
# ∂L/∂b = (2/N) * Σ(y_pred - y_true)
```

---

### Day 3–4 ｜ 实现完整训练循环

**实践目标**
- [ ] 💻 L3 能默写 ⭐：完整线性回归训练

```python
w, b = 0.0, 0.0
lr = 0.01
losses = []

for epoch in range(500):
    y_pred = w * x + b
    loss = np.mean((y_pred - y_true) ** 2)
    losses.append(loss)

    dw = np.mean(2 * (y_pred - y_true) * x)
    db = np.mean(2 * (y_pred - y_true))

    w -= lr * dw
    b -= lr * db
```

- [ ] 💻 L3 能默写：画 Loss 下降曲线

```python

import torch.nn as nn

class MLP(nn.Module):
    def __init__(self, in_dim, hid_dim, out_dim):
        super().__init__()
        self.fc1 = nn.Linear(in_dim, hid_dim)
        self.fc2 = nn.Linear(hid_dim, hid_dim)
        self.fc3 = nn.Linear(hid_dim, out_dim)

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = torch.relu(self.fc2(x))
        return self.fc3(x)    # 回归：最后一层不加激活
```

## Week 5 · PyTorch 训练循环与诊断

> 上周学了 PyTorch 的零件（Tensor / Autograd / Module / DataLoader），

### Day 1 ｜ 训练循环 5 步模板

**理论目标**
- [ ] 🧠 L2 理解 ⭐：训练循环每步含义

| 步骤 | 代码 | 含义 |
|------|------|------|
| ① 前向传播 | `pred = model(x)` | 数据过模型 |
| ② 算损失 | `loss = criterion(pred, y)` | 预测 vs 真实 |
| ③ 清旧梯度 | `optimizer.zero_grad()` | 清上一轮残留 |
| ④ 反向传播 | `loss.backward()` | 自动算梯度 |
| ⑤ 更新参数 | `optimizer.step()` | 沿梯度更新 |

- [ ] 🧠 L2 理解：为什么必须 `zero_grad()`——PyTorch 默认累加梯度

**实践目标**
- [ ] 💻 L3 能默写 ⭐⭐：完整训练循环

```python
model = MLP(1, 64, 1)
criterion = nn.MSELoss()
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)

for epoch in range(1000):
    pred = model(x_train)
    loss = criterion(pred, y_train)
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()

    if epoch % 100 == 0:
        print(f"Epoch {epoch}: Loss={loss.item():.6f}")
```

**⚠️ 极其重要**：这段代码贯穿全部后续学习。建议每天默写一遍，连写 3 天。

```python
class EarlyStopping:
    def __init__(self, patience=10):
        self.patience = patience
        self.counter = 0
        self.best = float('inf')
    def should_stop(self, val_loss):
        if val_loss < self.best:
            self.best = val_loss
            self.counter = 0
        else:
            self.counter += 1
        return self.counter >= self.patience
```

## Week 7 · autograd.grad() 求导数

### Day 1 ｜ backward() vs autograd.grad()

**理论目标**
- [ ] 🧠 L2 理解 ⭐：两种求导方式的区别

| | `loss.backward()` | `torch.autograd.grad()` |
|---|---|---|
| 用途 | 算 Loss 对 model.parameters 的梯度 | 算**任意输出**对**任意输入**的导数 |
| 场景 | 训练（更新参数） | 物理约束（算 du/dx 等） |
| 梯度存在哪 | 存到 `param.grad` | 直接返回 |

- [ ] 🧠 L2 理解：物理约束需要 `autograd.grad()`——比如网络输出位移 u(x)，你要算 ε = du/dx

**实践目标**
- [ ] 💻 L3 能默写 ⭐：

```python
x = torch.tensor([1.0, 2.0, 3.0], requires_grad=True)
y = x ** 2 + 3 * x     # y = x² + 3x

dydx = torch.autograd.grad(
    outputs=y, inputs=x,
    grad_outputs=torch.ones_like(y),
    create_graph=True
)[0]
print(dydx)   # → [5.0, 7.0, 9.0]  即 dy/dx = 2x+3 ✓
```

- [ ] 💻 L2 能照写：理解每个参数的含义
  - `grad_outputs=torch.ones_like(y)`：对向量输出求导必须指定
  - `create_graph=True`：保留计算图，允许后续再求导

## Week 13 · 消息传递与 GraphSAGE

### Day 1–2 ｜ 消息传递范式

**理论目标**
- [ ] 🧠 L2 理解 ⭐⭐：GNN 的统一框架

```
每个节点每一层做三步：
1. Message：每个邻居发一条消息
2. Aggregate：收集所有邻居消息，聚合（sum/mean/max）
3. Update：用聚合结果更新自身特征
```

- [ ] 🧠 L2 理解：GCN 是特例——消息=邻居特征×权重，聚合=加权平均

```python
from torch_geometric.nn import MessagePassing

class SimpleMP(MessagePassing):
    def __init__(self, in_ch, out_ch):
        super().__init__(aggr='mean')
        self.lin = nn.Linear(in_ch, out_ch)
    def forward(self, x, edge_index):
        return self.propagate(edge_index, x=x)
    def message(self, x_j):
        return self.lin(x_j)
```

```python
class EdgeAwareMP(MessagePassing):
    def __init__(self, node_dim, edge_dim, out_dim):
        super().__init__(aggr='mean')
        self.node_mlp = nn.Linear(node_dim, out_dim)
        self.edge_mlp = nn.Linear(edge_dim, out_dim)
        self.combine  = nn.Linear(out_dim*2, out_dim)
    def forward(self, x, edge_index, edge_attr):
        return self.propagate(edge_index, x=x, edge_attr=edge_attr)
    def message(self, x_j, edge_attr):
        nf = self.node_mlp(x_j)
        ef = self.edge_mlp(edge_attr)
        return torch.relu(self.combine(torch.cat([nf, ef], dim=-1)))
```
