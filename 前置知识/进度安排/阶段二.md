### Day 3–4 ｜ 不规则网格 → 图（升级版 mesh_to_pyg）

**理论目标**

```python
def mesh_to_graph_v2(nodes, elements, node_features=None, node_labels=None):
    """
    支持三角形和四边形单元的通用 mesh → graph 转换
    nodes: [N, 2 or 3] 坐标
    elements: list of arrays, 每个 array 是一个单元的节点索引
    """
    # 1. 提取边（从单元连接关系）
    edge_set = set()
    for elem in elements:
        n = len(elem)
        for i in range(n):
            for j in range(i+1, n):
                a, b = int(elem[i]), int(elem[j])
                edge_set.add((a, b))
                edge_set.add((b, a))

    src, dst = zip(*edge_set) if edge_set else ([], [])
    edge_index = torch.tensor([list(src), list(dst)], dtype=torch.long)

    # 2. 边特征（相对坐标 + 距离）
    edge_attr = []
    for s, d in zip(src, dst):
        diff = nodes[d] - nodes[s]
        dist = np.linalg.norm(diff)
        edge_attr.append(list(diff) + [dist])
    edge_attr = torch.tensor(edge_attr, dtype=torch.float)

    # 3. 节点特征
    if node_features is None:
        x = torch.tensor(nodes, dtype=torch.float)
    else:
        x = torch.tensor(
            np.column_stack([nodes, node_features]), dtype=torch.float
        )

    # 4. 标签
    y = torch.tensor(node_labels, dtype=torch.float) if node_labels is not None else None

    return Data(x=x, edge_index=edge_index, edge_attr=edge_attr, y=y)
```

```python
def check_dataset(dataset):
    for i, data in enumerate(dataset):
        # 检查 NaN
        assert not torch.isnan(data.x).any(), f"Sample {i}: NaN in features"
        assert not torch.isnan(data.y).any(), f"Sample {i}: NaN in labels"
```

### Day 3–4 ｜ 全局归一化 vs 样本归一化

**理论目标**
- [ ] 🧠 L2 理解 ⭐：图数据集的归一化有两种策略

| 策略 | 做法 | 适用场景 |
|------|------|---------|
| 全局归一化 | 用整个训练集的统计量（mean/std） | 所有样本的数值范围差不多 |
| 样本归一化 | 每个图独立归一化 | 不同样本的数值范围差异大 |

- [ ] 🧠 L2 理解：工程仿真中通常用**全局归一化**——因为不同样本间的材料参数有可比性

**实践目标**
- [ ] 💻 L4 能魔改 ⭐：为图数据集实现全局 Z-Score

```python
class GraphDatasetNormalizer:
    def fit(self, dataset):
        """遍历所有图，计算全局 mean 和 std"""
        all_x = torch.cat([d.x for d in dataset], dim=0)
        all_y = torch.cat([d.y for d in dataset], dim=0)
        self.x_mean, self.x_std = all_x.mean(0), all_x.std(0) + 1e-8
        self.y_mean, self.y_std = all_y.mean(0), all_y.std(0) + 1e-8

    def transform(self, dataset):
        for d in dataset:
            d.x = (d.x - self.x_mean) / self.x_std
            d.y = (d.y - self.y_mean) / self.y_std
        return dataset

    def inverse_y(self, y):
        return y * self.y_std + self.y_mean
```

## Week 23 · 编码器——把 FEM 数据编码为图

### Day 1–2 ｜ 编码器的设计思想

**理论目标**
- [ ] 🧠 L2 理解 ⭐：编码器的作用——把原始 FEM 数据（不同类型、不同维度）统一映射到同一个特征空间

```
原始数据（维度不统一）          编码后（统一维度 hidden_dim）
─────────────────          ─────────────────────
节点坐标 [N, 2/3]    ──→    节点嵌入 [N, hidden_dim]
材料参数 [N, k]       ──→    （融合到节点嵌入中）
边特征   [E, d]       ──→    边嵌入   [E, hidden_dim]
```

- [ ] 🧠 L2 理解：编码器通常由几个小 MLP 组成——分别处理节点特征和边特征，再拼接/融合
- [ ] 🧠 L2 理解：`hidden_dim` = 潜在特征空间的维度，是一个重要超参数

**实践目标**
- [ ] 💻 L4 能魔改 ⭐：实现编码器

```python
class FEMEncoder(nn.Module):
    def __init__(self, node_in_dim, edge_in_dim, hidden_dim):
        super().__init__()
        self.node_encoder = nn.Sequential(
            nn.Linear(node_in_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
        )
        self.edge_encoder = nn.Sequential(
            nn.Linear(edge_in_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
        )

    def forward(self, node_features, edge_attr):
        node_emb = self.node_encoder(node_features)  # [N, hidden]
        edge_emb = self.edge_encoder(edge_attr)       # [E, hidden]
        return node_emb, edge_emb
```

## Week 24 · 消息传递中的边更新

### Day 1–2 ｜ 同时更新节点和边

**理论目标**
- [ ] 🧠 L2 理解 ⭐：第一阶段的消息传递只更新了节点特征，边特征始终不变
- [ ] 🧠 L2 理解：更强的架构——每一层同时更新节点和边

```
一层的流程：
1. 用当前节点特征和边特征生成消息
2. 聚合消息更新节点特征
3. 用更新后的节点特征更新边特征
```

**实践目标**
- [ ] 💻 L4 能魔改 ⭐⭐：实现同时更新节点和边的消息传递层

```python
class NodeEdgeUpdateLayer(MessagePassing):
    def __init__(self, node_dim, edge_dim):
        super().__init__(aggr='mean')
        self.msg_mlp = nn.Sequential(
            nn.Linear(node_dim + edge_dim, node_dim), nn.ReLU()
        )
        self.node_update = nn.Sequential(
            nn.Linear(node_dim * 2, node_dim), nn.ReLU()
        )
        self.edge_update = nn.Sequential(
            nn.Linear(node_dim * 2 + edge_dim, edge_dim), nn.ReLU()
        )

    def forward(self, x, edge_index, edge_attr):
        # 1. 节点更新
        aggr = self.propagate(edge_index, x=x, edge_attr=edge_attr)
        x_new = self.node_update(torch.cat([x, aggr], dim=-1))

        # 2. 边更新
        src, dst = edge_index
        edge_attr_new = self.edge_update(
            torch.cat([x_new[src], x_new[dst], edge_attr], dim=-1)
        )
        return x_new, edge_attr_new

    def message(self, x_j, edge_attr):
        return self.msg_mlp(torch.cat([x_j, edge_attr], dim=-1))
```

### Day 3–4 ｜ 多层堆叠 + 残差

```python
class MessagePassingStack(nn.Module):
    def __init__(self, node_dim, edge_dim, n_layers=3):
        super().__init__()
        self.layers = nn.ModuleList([
            NodeEdgeUpdateLayer(node_dim, edge_dim) for _ in range(n_layers)
        ])

    def forward(self, x, edge_index, edge_attr):
        for layer in self.layers:
            x_new, edge_attr_new = layer(x, edge_index, edge_attr)
            x = x + x_new              # 残差
            edge_attr = edge_attr + edge_attr_new  # 边残差
        return x, edge_attr
```

```python
class ElementAggregator(nn.Module):
    def __init__(self, node_dim, elem_dim):
        super().__init__()
        self.mlp = nn.Sequential(
            nn.Linear(node_dim, elem_dim), nn.ReLU(),
            nn.Linear(elem_dim, elem_dim),
        )

    def forward(self, node_features, element_connectivity):
        """
        node_features: [N_nodes, node_dim]
        element_connectivity: list of lists, 每个元素是该单元的节点索引
        """
        elem_feats = []
        for node_ids in element_connectivity:
            # 取该单元所有节点的特征平均
            elem_feat = node_features[node_ids].mean(dim=0)
            elem_feats.append(elem_feat)
        elem_feats = torch.stack(elem_feats)      # [N_elems, node_dim]
        return self.mlp(elem_feats)                # [N_elems, elem_dim]
```

```python
def build_scatter_matrix(elements, n_nodes):
    """构建 [N_elems, N_nodes] 的稀疏聚合矩阵"""
    rows, cols = [], []
    for eid, nids in enumerate(elements):
        for nid in nids:
            rows.append(eid)
            cols.append(nid)
    values = torch.ones(len(rows))
    C = torch.sparse_coo_tensor(
        torch.tensor([rows, cols]), values,
        (len(elements), n_nodes)
    )
    # 归一化（每行除以该单元的节点数）
    row_sum = torch.sparse.sum(C, dim=1).to_dense().unsqueeze(1)
    return C, row_sum

# 使用
C, row_sum = build_scatter_matrix(elements, n_nodes)
elem_feats = torch.sparse.mm(C, node_features) / row_sum
```

```python
class FEMGraphNet(nn.Module):
    def __init__(self, node_in, edge_in, hidden, disp_out, stress_out):
        super().__init__()
        self.encoder = FEMEncoder(node_in, edge_in, hidden)

    def forward(self, data, elements):

        node_emb, edge_emb = self.processor(node_emb, data.edge_index, edge_emb)
        # 单元聚合
        elem_emb = self.elem_agg(node_emb, elements)
        # 解码
        disp, stress = self.decoder(node_emb, elem_emb)
        return disp, stress
```

## Week 28 · GNN + 几何方程约束

### Day 1–2 ｜ 在 GNN 中嵌入几何方程

**理论目标**
- [ ] 🧠 L2 理解 ⭐⭐：几何方程——应变 = 位移的导数

```
εxx = ∂ux/∂x,  εyy = ∂uy/∂y,  εxy = ½(∂ux/∂y + ∂uy/∂x)
```

- [ ] 🧠 L2 理解：GNN 预测位移 → 用数值微分近似计算应变 → 检查是否满足几何方程
- [ ] 🧠 L2 理解：在图上做数值微分——用相邻节点的位移差除以距离来近似偏导数

**实践目标**
- [ ] 💻 L4 能魔改 ⭐：在图上近似计算应变

```python
def compute_strain_on_graph(u_pred, data):
    """用相邻节点的位移差近似应变"""
    src, dst = data.edge_index
    dx = data.x[dst, 0] - data.x[src, 0]   # Δx
    dy = data.x[dst, 1] - data.x[src, 1]   # Δy
    dux = u_pred[dst, 0] - u_pred[src, 0]  # Δux
    duy = u_pred[dst, 1] - u_pred[src, 1]  # Δuy
    dist = torch.sqrt(dx**2 + dy**2 + 1e-8)

    # 近似偏导数
    dux_dx = dux * dx / (dist**2 + 1e-8)
    duy_dy = duy * dy / (dist**2 + 1e-8)

    return dux_dx, duy_dy  # εxx 和 εyy 的近似
```

## Week 29 · GNN + 本构方程约束

### Day 1–2 ｜ 嵌入线弹性本构方程

**理论目标**
- [ ] 🧠 L2 理解 ⭐⭐：线弹性本构方程（胡克定律的 2D 平面应力形式）

```
σxx = E/(1-ν²) × (εxx + ν·εyy)
σyy = E/(1-ν²) × (εyy + ν·εxx)
σxy = E/(2(1+ν)) × εxy
```

- [ ] 🧠 L2 理解：GNN 预测位移 → 算应变 → 用本构方程算应力 → 检查和 FEM 应力是否一致
- [ ] 🧠 L2 理解：这就是"嵌入本构方程"的核心——用已知物理关系约束网络输出

**实践目标**
- [ ] 💻 L4 能魔改 ⭐⭐：嵌入本构方程

```python
def constitutive_law(eps_xx, eps_yy, eps_xy, E, nu):
    """2D 平面应力本构方程"""
    coeff = E / (1 - nu**2)
    sigma_xx = coeff * (eps_xx + nu * eps_yy)
    sigma_yy = coeff * (eps_yy + nu * eps_xx)
    sigma_xy = E / (2 * (1 + nu)) * eps_xy
    return sigma_xx, sigma_yy, sigma_xy
```

```python
def balanced_weights(losses):
    """让每项 loss 的量级大致相同"""
    with torch.no_grad():
        magnitudes = [l.item() for l in losses]
        total = sum(magnitudes)
        weights = [total / (len(losses) * m + 1e-8) for m in magnitudes]
    return weights
```
