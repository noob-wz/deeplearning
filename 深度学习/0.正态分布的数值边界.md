**数学原点：考试成绩的“优良中差”比例**

* **一句话是什么**：标准差=1，意思是“**大多数人**”在 -1 到 1 之间，但总有少数“尖子生”和“极差生”会跑到外面去。
* **为什么从这里出发**：用均匀分布（`torch.rand`）去类比的话，均匀分布就像一个长方形的盒子，说 0 到 1 之间，就绝不会有一点越界。但正态分布（`torch.randn`）是一座“没有绝对边界的山丘”，越往两边人越少，但永远不可能彻底没人。

学完后你能做到：看到 `torch.randn`，脑海里能精确预估里面每个数字出现的概率大小。

需要掌握的概念：
在统计学里，有一个著名的经验法则专门用来回答你的疑问：
▸ 距离平均值 **1个标准差** 范围内（即 -1 到 1 之间）：包含了约 **68%** 的数据。也就是说，超过一半的数字确实是在这个范围里浮动！
▸ 距离平均值 **2个标准差** 范围内（即 -2 到 2 之间）：包含了约 **95%** 的数据。
▸ 距离平均值 **3个标准差** 范围内（即 -3 到 3 之间）：包含了约 **99.7%** 的数据。

**CV实战意义**：
如果你的权重矩阵有 10000 个参数。用 `torch.randn` 初始化时：

* 大约有 6800 个参数会在 -1 到 1 之间。
* 大约有 3200 个参数会跑到 -1 到 1 的“外面”。
* 甚至可能有二三十个“极其幸运”的参数，数值超过了 2 或者小于 -2！

推荐学习资源：     
📝 **动手破除迷信**：打开 Python，我们用代码直接数一数！
```python
import torch

# 生成 10000 个正态分布的随机数
data = torch.randn(10000)

# 看看有多少个数是在 -1 到 1 之间
# data.abs() <= 1 会找出绝对值小于等于 1 的数
in_range = (data.abs() <= 1).float().mean()
print(f"在 -1 到 1 之间的比例: {in_range * 100:.2f}%")
# 运行结果一定会非常接近 68%！
```
